## Claim

Seven gaps no existing tool covers: (1) no runtime hook validation — all do static analysis only, never verify hooks execute, (2) no integrated cost/token tracking, (3) no multi-project aggregation, (4) no team setup diff/standardization, (5) no CLAUDE.md quality scoring beyond length — none check for contradictory instructions or outdated rules, (6) no MCP tool call auditing — configured vs actually used, (7) no hook execution tracing. These are claudoctor's differentiation opportunities.
