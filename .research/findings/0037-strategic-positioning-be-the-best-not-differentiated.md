## Claim

Claudoctor should not carve a niche to avoid competition. It should be the definitive Claude Code diagnostic tool — subsuming everything cc-health-check (20 static checks), second-brain-health-check (45 opinionated checks), and Waza /health (qualitative skill) do, plus everything they don't: runtime probing, security CVE detection, disk/process hygiene, CLAUDE.md quality analysis, MCP server liveness, hook execution tracing. The model is flutter doctor — it didn't avoid competing with xcode-select, it absorbed that check and added 50 more. Claudoctor wins by being comprehensive, accurate, fast, methodology-agnostic, and available everywhere (CLI + MCP + CI).
