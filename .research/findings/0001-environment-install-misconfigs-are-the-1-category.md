## Claim

Most common Claude Code misconfigs are environment-level: ANTHROPIC_API_KEY not exported (set but not sourced), PATH not updated (command not found), expired OAuth tokens mistaken for outage, npm EACCES permission issues. These should be claudoctor's first checks.
